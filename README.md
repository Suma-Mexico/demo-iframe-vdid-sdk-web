# Integration of the vdid-sdk-web

## Getting Started

-   `pnpm install` - Install npm packages

-   `pnpm dev` - Starts a dev server at http://localhost:3000/

-   `pnpm build` - Builds for production, emitting to `dist/`. Prerenders app to static HTML

-   `pnpm preview` - Starts a server at http://localhost:3000/ to test production build locally
